## 5 · Neural-net tabular model — **NEW TOP OPEN BET**

### Why this jumped to the top

`brief.md:74`: synthetic data was generated by a deep-learning model
trained on the Irrigation Prediction dataset. The 2026-04-21 EDA
(plots/eda/dgp_residuals.html) provides three independent pieces of
evidence that labels are a *deterministic function* of the full
feature vector, not rule + noise:

1. **Zero exact feature-vector duplicates** in 630k rows — consistent
   with a VAE / diffusion sampler, not with a rule + IID flip.
2. **At score=3 (4,899 flips / 102k rows), non-rule features differ
   significantly between flipped and non-flipped rows**:
   `Previous_Irrigation_mm` d=+0.107 (p=5e-14), `Humidity` d=+0.076
   (p=8e-8). Pure noise → d≈0 on every non-rule feature.
3. **Per-cell majority predictor** on the 64 discrete rule-cells
   scores raw 0.98384 / bal 0.95983 — effectively identical to the
   rule. Only 1 cell has a synthetic-majority different from the
   rule. So the noise is NOT cell-level; it's within-cell, driven
   by continuous position + non-rule features.

Implication: **no irreducible noise floor; the ceiling is 100 %.**
Axis-aligned trees are structurally handicapped — a NN's decision
boundary is a smooth manifold that trees need many splits to
approximate. A tabular NN (MLP / TabNet / a small ResNet with
periodic activations) is the closest structural match to the DGP.

### Plan

- Standardise numerics (z-score with train stats), one-hot / embedding
  the 8 categoricals, pass through a 3-layer MLP (e.g. 256-128-64
  with ReLU + dropout 0.1), softmax output, cross-entropy loss.
- 5-fold stratified CV with the same seed=42 splits so the OOF number
  is directly comparable to 0.97271 LGBM+DGP baseline.
- Include the 15 DGP-derived columns as inputs — they're a free
  lift the NN can modulate.
- Apply the same log-bias coord-ascent on OOF probs.
- Target: break 0.975 tuned OOF. A single MLP that matches the pack
  would sit at ~0.9800; beating it requires blending.

Expected delta: the NN is untried, so upside is genuinely unknown —
but structurally it's the best-matched model family to the DGP.
Downside is setup cost (~2 hours).

## 9 · Ordinal-aware loss / Medium↔High sample-weighting

Confusion mass lives at Medium↔High. LGBM `multiclassova` with
weights that upweight adjacent-class misclassifications. The 2026-04-21
EDA confirms flips are **always to the adjacent class** (score=3
→ Medium, never → High), so an ordinal model is a structural match
to the noise pattern. Expected +0.001 or null.
